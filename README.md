## Installation
```sh
mkdir -p ~/.config/ulauncher/user-themes
git clone https://github.com/BartoszBartmanski/Ulauncher-Essential-Dark-Theme-Green.git \
  ~/.config/ulauncher/user-themes/Essential-Dark-Theme
```
